# README.md

## Server

**To run server:**

1. **cd** server

2. **pnpm**/**npm**/**yarn** install

3. **pnpm**/**npm**/**yarn** run start

4. App will run on **localhost:3010**

## Client

**To run web app:**

1. **cd** client

2. **pnpm**/**npm**/**yarn** install

3. **pnpm**/**npm**/**yarn** run start

4. App will run on **localhost:5173**
